# ============================================================================
# Installation Notes:
# ============================================================================
# 
# 1. For local development with CUDA support:
#    pip install torch==2.6.0+cu124 -f https://download.pytorch.org/whl/torch_stable.html
#
# 2. For Databricks deployment:
#    - Install via notebook: %pip install -r requirements.txt
#    - Or via cluster libraries UI
#    - serverless_gpu is pre-installed on Databricks Serverless GPU compute
#
# 3. Minimum Python version: 3.10+
#
# 4. For CPU-only installation (no CUDA):
#    pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
#
# 5. To check installed versions:
#    pip list | grep -E "torch|ultralytics|mlflow|opencv"
#
